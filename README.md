# Dev Container for OOP (C++)

This is a personal dev container converted from Java to C++ for practicing Object-Oriented Programming tasks.

## What's Included:

### C++ Development Tools:
- **GCC/G++** - GNU C++ compiler (build-essential)
- **GDB** - GNU Debugger
- **CMake** - Build system generator
- **Ninja** - Fast build system
- **Clang Format** - Code formatter
- **Clang Tidy** - Static analysis tool
- **Cppcheck** - Static code analysis
- **Valgrind** - Memory debugging tool

### VS Code Extensions:
- **C/C++** (ms-vscode.cpptools) - IntelliSense, debugging, and code browsing
- **C/C++ Extension Pack** - Complete C++ development experience
- **CMake Tools** - CMake integration

## Setup:

The container automatically installs all C++ development tools on creation. The setup includes:
- C++17 standard support
- IntelliSense configuration for Linux GCC
- Code formatting with Clang Format
- Error squiggles and diagnostics

## Usage:

This container is ready for C++ OOP practice. You can:
- Create `.cpp` and `.h` files for your exercises
- Use CMake for building projects
- Debug with GDB through VS Code
- Format code with Clang Format
- Analyze code with Clang Tidy and Cppcheck
